request-header-parser-microservice 1.0.0

  • User Story: I can get the IP address, language and operating system for my browser.

API Usage

/whoami

Returns a JSON object containing the users ipaddress, language, and software.

For example:

https://header-parser-logan.herokuapp.com/whoami

would return

{"ipaddress":"123.45.678.901","language":"en-US","software":"Windows NT 10.0; Win64; x64"}
